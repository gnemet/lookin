Mi az a vektor?

A kérdés és a dokumentumok szövegét egy AI modell 1024-dimenziós számsorrá alakítja. A hasonló tartalmak közel kerülnek egymáshoz a vektortérben — mint egy 1024-dimenziós térkép.

Az MCP szerver

A "hídprogram" — az MCP szerver

Mi az MCP?

A Model Context Protocol (Anthropic szabvány) lehetővé teszi, hogy a Claude külső adatforrásokat és eszközöket hívjon meg strukturáltan — mint egy API, de az AI-ra szabva.

  • Claude Code CLI elindítja a Python szervert (.mcp.json)
  • Kommunikáció: stdio (standard bemenet/kimenet)
  • GPG vault: .env.gpg → .env automatikusan feloldódik
  • Claude "látja" az összes eszközt és leírást

Hogyan dönti el Claude, mit hívjon?

Minden eszköznek van egy szöveges leírása. Claude értelmezi a kérdést, és eldönti, melyik eszközt (vagy kombinációjukat) kell meghívni.

Biztonság

Háromrétegű biztonsági modell

1. réteg

GPG Vault

  • .env.gpg — titkosítva a git-ben
  • VAULT_PASS — csapatkulcs (CI/CD változó)
  • Telepítéskor automatikusan feloldódik
  • DB jelszavak, PAT tokenek, host-ok — mind vaultban
  • Maintainer zárolja vissza és pusholja
2. réteg

Row Level Security (RLS)

  • A RAG adatbázison PostgreSQL RLS engedélyezve
  • Az it-akos role csak a saját tenant adatait látja
  • Más tenants adatai fizikailag el vannak választva
  • Applikáció-szintű szűrés nem szükséges — DB garantálja
  • Pipeline-írás: root (RLS megkerüli)
3. réteg

SECURITY DEFINER

  • DWH-ban az it-akos role csak EXECUTE jogot kap
  • Közvetlen tábla SELECT jog nincs
  • Minden adat SECURITY DEFINER funkciókon keresztül
  • Az admin szabja meg, mit lát az applikáció

PAT tokenek (Jira / Confluence)

  • Atlassian Personal Access Token — 90 nap érvényesség
  • Jelenleg: jövőbeli mély-fetch eszközökhöz fenntartva
  • A mai 32 eszköz nélkülük is működik
  • Rotáláskor: vault unlock → szerkeszt → lock → push

Cross-platform Windows support

  • ${PYTHON:-python3} a .mcp.json-ban
  • Linux/WSL: python3 alapértelmezett
  • Windows: PYTHON=py egyszer beállítva
  • Ugyanaz a commit, minden platformon fut
