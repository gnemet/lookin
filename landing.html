100% Static · No Backend Required

LookIn 🔍

A YAML-controlled, interactive architecture viewer with hand-drawn Mermaid diagrams. Drill down from business flows to table columns — across multiple projects.

🏗️ View JiraDa Architecture ⭐ GitHub Repository
Features

Everything you need to explore architecture

✏️

Hand-Drawn Style

Chalkboard-aesthetic PNG diagrams with Mermaid's handDrawn theme for an approachable, sketch-like look.

📋

YAML-Controlled

Define layers, drill-downs, click regions, and sources in a single YAML config file. No code changes needed.

🗺️

TOC Sidebar

Collapsible category groups with curated navigation. Jump to any layer instantly with keyboard shortcuts.

📖

Doc Panel

Right-side panel slides in for catalog data, markdown documentation, or Mermaid source — with anchor scrolling.

🔗

Multi-Project

Wire multiple projects into one unified view. Color-coded sources show which repo each layer belongs to.

🔍

Drill-Down

Click nodes to zoom deeper: Business → Architecture → Schema → Columns. Pan, zoom, and navigate with keyboard.

🖼️

PNG Presentation Mode

Pre-rendered PNG diagrams with click regions for offline/LAN use — no mermaid.js required at runtime.

🌐

Bilingual (EN/HU)

Toggle between English and Hungarian with a header button. All layer titles and ToC groups are translatable.

⚡

GitHub Pages Ready

100% static — just open index.html. Auto-deployed via GitHub Actions. Works offline from any folder.

Quick Start

Up and running in seconds

terminal 
# Just open in browser
open index.html

# Or serve locally
python3 -m http.server 8000

# Regenerate docs from source projects
python3 generate_docs.py

# Regenerate PNG diagrams (requires Puppeteer)
./generate_png.sh
Project Structure

Clean, predictable layout

lookin/
├── index.html — Single-page viewer
├── app.js — Navigation, TOC, pan/zoom, rendering
├── style.css — Catppuccin dark theme
├── configs/ — YAML layer configs
│ └── jirada.yaml — Layer + drill-down mapping
├── layers/ — Mermaid diagrams per project
│ ├── jiramntr/ — DWH architecture (10 diagrams)
│ ├── johanna/ — AI chat architecture
│ ├── mcp-forge/ — RAG pipeline
│ └── aichat/ — AI module
├── docs/ — Markdown docs per layer
├── catalogs/ — JSON table catalogs
├── generate_docs.py — Auto-generate from source
└── generate_png.sh — Render MMD → PNG
